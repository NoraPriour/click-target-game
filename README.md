# Click Target Game

Mini jeu web où le joueur doit cliquer sur une cible le plus vite possible pour marquer des points.

## Objectif du projet

Ce projet permet de pratiquer :
- HTML, CSS et JavaScript
- Java avec Spring Boot
- une base de données avec utilisateurs et scores

## Fonctionnalités prévues

- Création de compte avec pseudo et mot de passe
- Connexion utilisateur
- Jeu de clic avec score et chronomètre
- Sauvegarde des scores
- Classement des meilleurs joueurs
